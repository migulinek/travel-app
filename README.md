# Travel App with Fake API in Symfony

## Tech Stack

- Docker
- Symfony 6
- React 18

### Installing & running

run docker and connect to container:

```
 docker compose up -d
```

### Ready up

click [http://localhost:3000/](http://localhost:3000/) in your browser
